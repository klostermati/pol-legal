PanelsOfLife

PanelsOfLife is a personal automated content pipeline that generates and publishes short-form AI-generated watercolor comic videos to social media channels owned by the operator.

Terms of Service

Last updated: June 23, 2026

PanelsOfLife is a personal, non-commercial automated publishing tool. It is not a public-facing product or service offered to third parties.

  • The tool operates exclusively on social media accounts owned and controlled by the operator.
  • No third-party users interact with or have access to this tool.
  • All content posted via this tool is generated and owned by the operator.
  • The operator complies with the terms of service of each platform the tool posts to (TikTok, Instagram, YouTube).
  • All AI-generated content is disclosed to each platform using the available API flags (e.g., is_aigc: true on TikTok) and account-level disclosures where available.

Privacy Policy

Last updated: June 23, 2026

PanelsOfLife does not collect, store, or process personal data from any third parties.

  • No user data collected. This tool has no end users. It operates solely on behalf of the operator.
  • API credentials. OAuth tokens used to authenticate with social media platforms are stored on the operator's private infrastructure (local machine or private AWS environment) and are never shared.
  • No analytics or tracking. No cookies, analytics, or tracking mechanisms are used.
